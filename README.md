## Getting Started

1. Install dependencies

   ```bash
   yarn
   ```

2. Fire up the server and watch files

   ```bash
   npm run dev
   ```

## Deployment

1. Install the serverless.js CLI

   ```bash
   npm install -g serverless
   ```

2. Deploy!

   ```bash
   serverless
   ```
